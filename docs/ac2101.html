💼 Contexte de réalisation

Cadre : SAÉ 303 — Pépinière Multi-site

Description : Implémentation du routage d'une infrastructure d'entreprise distribuée (Siège social et succursale) simulant un transit opérateur sur le VLAN 800.

🛠️ Travail Réalisé & Justifications Techniques

Pour cette tâche, nous avons déployé le protocole de routage dynamique OSPF (Open Shortest Path First). Afin d'éviter d'inonder le réseau public de l'opérateur virtuel (VLAN 800) avec nos tables de routage internes, j'ai encapsulé les flux dans un tunnel VPN IPsec / GRE reliant le Siège et la succursale.

Commandes et validations clés :

  • Définition des réseaux de la pépinière dans le processus OSPF (router ospf 1).
  • Configuration des interfaces passives (passive-interface) vers les VLANs utilisateurs (Admin, Personnel, etc.) pour sécuriser et économiser la bande passante.
  • Vérification des tables avec la commande show ip route ospf pour valider l'apprentissage des routes de la succursale distante.
