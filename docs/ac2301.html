AC23.01 | Automatisation & Scripting

Automatiser l’administration système avec des scripts

← Retour à la compétence

Description de la Réalisation (SAÉ 303)

Dans la SAÉ 303 (Pépinière), l'un des besoins fondamentaux était de simplifier et d'automatiser l'accès aux ressources partagées pour les employés d'une entreprise lors de leur authentification.

Détails Techniques de la Preuve :

  • Script d'automatisation (PowerShell) : Écriture d'un script de connexion (Logon Script) destiné à mapper dynamiquement et automatiquement les lecteurs réseaux (partages de fichiers Windows) sur les postes clients selon l'identité de l'utilisateur.
  • Déploiement centralisé : Intégration du script PowerShell au sein d'une Unité d'Organisation (OU) et liaison via une Stratégie de Groupe (GPO) Active Directory sur Windows Server 2022.
  • Gestion des politiques d'exécution : Ajustement des GPO pour autoriser l'exécution transparente du script sans invites de sécurité intrusives sur les machines du domaine.
