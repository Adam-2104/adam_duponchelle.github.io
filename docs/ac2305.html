AC23.05 | Accès aux Données

Accéder à un ensemble de données depuis une application et/ou un site web

← Retour à la compétence

Description de la Réalisation (SAÉ 303)

Dans la SAÉ 303 (Pépinière), j'ai participé à la mise en œuvre d'un service web d'entreprise interrogeant un ensemble de données stocké de manière distribuée.

Détails Techniques de la Preuve :

  • Virtualisation par Conteneurs : Écriture et utilisation de descripteurs Docker Compose pour instancier l'infrastructure applicative.
  • Liaison Applicative : Déploiement conjoint d'un conteneur hébergeant un serveur web et d'un conteneur de Base de Données.
  • Haute Disponibilité : Implémentation d'une réplication de la base de données en mode lecture seule (réplicat) localisée sur le site de la succursale, permettant à l'application web distante d'accéder localement aux données de l'entreprise de manière performante.
